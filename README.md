# Square Fall - Hyper casual game for yandex

This game was created for the Yandex Games portal.
The game implements the following functions from the Yandex SDK:
-Leaderboard
-Monetization
-Copy Protection

-List of extensions used:
-defos
-defold-random
-defold-sharp-sprite
-monarch
-defold-yagames
-defsave
